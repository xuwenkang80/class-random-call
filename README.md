# 班级随机点名

#### 介绍
基于random和tkinter库编写的py程序。可支持txt导入文件

#### 软件架构
软件架构说明


#### 安装教程

1.  在本地环境安装Python3.7及以上版本
2.  下载main.py文件

#### 使用说明

1.  在同一根目录下新建一个name.txt文件，每行输入一位同学的姓名
2.  运行main.py
3.  开始点名！
